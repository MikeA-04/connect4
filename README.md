# Connect Four
The game opens with a welcome screen. To start the game, press "Start."\
<img src="https://user-images.githubusercontent.com/69696262/162585905-da53054e-ac93-4206-b742-f751f3ff3df6.png" width=60% height=60%> <br><br>
\
There is a message at the top of the game window that tells which player's turn it is.\
<img src="https://user-images.githubusercontent.com/69696262/162585955-8e6984f0-9deb-4e38-889f-4e2d143fe8d1.png" width=60% height=60%> <br><br>
\
A player can reverse their move under the "Gameplay" menu. If they do, it is shown at the top as well as whose turn it is.\
<img src="https://user-images.githubusercontent.com/69696262/162585982-5220386f-82b6-4c7e-a746-3518be00b2a2.png" width=60% height=60%> <br><br>
\
If a player tries to make an invalid move, a message is displayed at the top letting the player know.\
<img src="https://user-images.githubusercontent.com/69696262/162585991-5dd5f390-6011-4abf-aa5d-1ee071cc32a9.png" width=60% height=60%> <br><br>
\
If a player wins, the four squares that fulfill the winning condition are highlighted for a few seconds. Then, a new screen is shown to let the players know who won.\
<img src="https://user-images.githubusercontent.com/69696262/162586010-986db104-481a-4d61-88ec-10559fe3ffd4.png" width=60% height=60%> <br><br>
\
There are different themes available under the "Themes" menu.\
The game can be restarted or closed at any time under the "Options" menu.\
There is a "Help" feature which opens another window with the rules also under the "Options" menu.\
<img src="https://user-images.githubusercontent.com/69696262/162586096-d46ba799-47a7-4fce-a199-7bb5b1c8e0e1.png" width=50% height=50%>
